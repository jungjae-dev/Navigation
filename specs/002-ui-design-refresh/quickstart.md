# Quickstart: 디자인 통일 리팩토링 검증

**Feature**: 002-ui-design-refresh | **Date**: 2026-06-07

본 작업은 시각·레이아웃 변경이므로 **시뮬레이터 시각 검증**으로 완료를 판정한다(plan Constitution Check IV 적응).

## 빌드 & 실행
```bash
# 시뮬레이터: iPhone 17 Pro (iOS 26)
xcodebuild -scheme Navigation -destination 'platform=iOS Simulator,name=iPhone 17 Pro' build
```
앱 실행 후 인스코프 화면을 순회한다: 홈/지도 → 검색 → 검색 결과 → 경로 미리보기 → 장소/대여소/정류장 상세 → 버스 노선 → 버스 시간표 → 설정.

## 검증 체크리스트 (성공 기준 매핑)

### 1. 통일성 (SC-001, SC-002, SC-004)
- [ ] 모든 인스코프 화면에서 강조색이 동일한 인디고 accent 하나로만 보인다.
- [ ] 같은 종류 버튼/카드/시트/셀/헤더가 화면 간 동일하게 보인다.
- [ ] 임시방편(튀는 색/폰트/간격) 흔적이 없다.

### 2. 액센트 절제 (SC-009)
- [ ] accent는 주요 액션·선택/활성·링크에만 보이고, 표면/본문/장식 아이콘은 중립색이다.

### 3. 시트 일원화 (SC-008)
- [ ] 모든 하단 시트가 그래버 + 다단계 detent + 불투명 솔리드 동일 패턴이다.
- [ ] 화면별 커스텀 시트/반투명 표면이 없다.

### 4. 라이트/다크 & 대비 (SC-003)
- [ ] 라이트/다크 전환 시 모든 화면이 통일·가독 유지.
- [ ] 본문 텍스트·액션 버튼 라벨이 WCAG AA(4.5:1) 이상. (대비 측정: 디자인 검수 도구 또는 색상 대비 계산)

### 5. Dynamic Type (SC-006)
- [ ] 설정 > 손쉬운 사용 > 글자 크기를 한 단계 이상 키운 뒤 인스코프 화면에서 겹침·정보 손실 잘림 0건.
- [ ] 셀/헤더가 큰 글자에 맞춰 성장(고정 height로 잘리지 않음).

### 6. 빈/로딩 상태 (SC-007)
- [ ] 검색 결과 없음 / 즐겨찾기·최근기록 없음 / 도착정보 없음 / 로딩 중이 공통 EmptyStateView로 일관 표시.

### 7. 플로팅 컨트롤 (FR-007b)
- [ ] 홈/지도 상시 노출 컨트롤이 최소화·그룹화되어 지도 가시영역이 넓어졌다(기능 접근은 유지).

### 8. 범위 회귀 (SC-005)
- [ ] 주행화면(턴 배너·하단 바·속도계·도착 팝업·가상주행) 모양·동작 변화 없음.
- [ ] DevTools·CarPlay 변화 없음.

## 스크린샷 비교
- 변경 전/후 동일 화면을 같은 조건(라이트·기본 글자)에서 캡처해 나란히 비교, "하나의 통일된 깔끔한 앱" 인지(SC-004) 정성 합격 확인.
- 가능하면 애플 지도 동등 화면과 병치해 "심플·정돈" 방향성 점검.

## 회귀 안전망 (선택)
- accent/typography 토큰 매핑·대비에 대한 경량 단위 테스트(Swift Testing) 보강 가능.
